Функции all() и any() в Julia

Функция all()

Функция all() проверяет, выполняется ли условие для всех элементов коллекции.

all(x -> x > 0, [1, 2, 3])  # true
all(x -> x > 0, [1, -2, 3]) # false

Функция any()

Функция any() проверяет, выполняется ли условие хотя бы для одного элемента.

any(x -> x > 0, [-1, -2, 3]) # true
any(x -> x > 0, [-1, -2, -3]) # false

Сравнение

  • all() — все элементы должны удовлетворять условию
  • any() — достаточно одного элемента

Пример

nums = [2, 4, 6]
println(all(x -> x % 2 == 0, nums)) # true
println(any(x -> x % 2 == 0, nums)) # true
